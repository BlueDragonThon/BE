# 📙 서비스 목적

- 스마트폰 보유율은 76.6%로 시간이 지날 수록 증가하는 추세지만, 노인의 67.2%는 정보화 사회 적응에 어려움을 느끼고 있음
- 노인 대학에 대한 인지도가 높지 않을 뿐만 아니라, 노인 대학의 신청 절차는 접근성이 낮음.
- 노인 대학의 홍보를 강화하고, 온라인 및 오프라인 신청 절차를 간소화하여 보다 쉽게 접근할 수 있도록 개선
- 교육 프로그램을 확대함으로써 노인들이 정보화 사회에 적응할 수 있도록 지원
